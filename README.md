# Pomodoro-Timer
Arduinoを使ってポモドーロタイマーを作成しました．

## ポモドーロタイマーとは
25分作業，5分休憩のサイクルを繰り返すことで，持続可能な集中を得られるというサイクルのことです．

## プログラム
### Pomodoro-Timer.ino
ポモドーロタイマーの本プログラムです．
Arduinoに書き込みます．

### Bottun-Test.ino
タクトスイッチのテストプログラムです．
参考URL：https://deviceplus.jp/arduino/arduino_f06/

### Display-Test.ino
ディスプレイのテストプログラムです．
参考URL：https://www.petitmonte.com/robot/howto_aqm1602xa_rn_gbw.html

### Motor-Test.ino
サーボモータのテストプログラムです．
参考URL：https://miraluna.hatenablog.com/entry/penguin
